# Markdown

一种可以使用普通文本编辑器编写的标记语言，通过简单的标记语法，它可以使普通文本内容具有一定的格式。

# 标题

# 标题1
## 标题2
### 标题3

# 效果

# 标题1

# 标题2

# 标题3

+ 1
+ 2
+ 3

# vuepress中封装的Markdown

# 需要在config.js中配置

plugins: [
      'vuepress-plugin-container',
      {
        type: 'tip',
        defaultTitle: {
          '/': '提示',
          '/zh/': '提示',
        },
      },
    ]
  ]

# tip

::: tip
This is a tip
:::

::: warning
This is a warning
:::

::: danger
This is a dangerous warning
:::

# 效果

提示

This is a tip

注意

This is a warning

警告

This is a dangerous warning
